# Future Use - CSS Files

This folder contains CSS files that are **not currently in use** in the ZamboDorm project.
## If You Need These
If you plan to reactivate any admin features or need these styles:
1. Copy the file back to the parent `css` folder
2. Update the HTML file references from `../css/filename.css` to point to the new location
3. Test thoroughly to ensure compatibility

---
**Note:** The active/working CSS files are in the parent `css` folder.
