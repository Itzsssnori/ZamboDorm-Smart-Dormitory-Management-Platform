Properties Management
How do I add a new property to the system?
Go to Properties → Add New Property. Fill in details including property name, address, number of floors, total rooms, and amenities. Assign administrators and set room pricing. Click Save to create the property in the system.
Was this helpful?
How do I manage room availability?
Go to Properties → Select Property → Rooms. You can view all rooms, their status (occupied, available, maintenance), and details. To mark a room as unavailable, click the room and select Set to Maintenance or Unavailable. This prevents new tenants from booking it.
Was this helpful?
How do I update property amenities or details?
Go to Properties → Select Property → Edit Details. You can update amenities, pricing, photos, descriptions, and contact information. Changes take effect immediately and are visible to tenants and administrators.
Was this helpful?
How do I assign admins to a property?
Go to Properties → Select Property → Administrators. Click Add Admin and select from the list of available administrators. Admins can manage tenants, rooms, and complaints for this property. Remove admins by clicking Remove next to their name.
Was this helpful?
How do I view property statistics and reports?
Go to Overview to see system-wide statistics. For property-specific data, go to Properties → Select Property → Analytics. You can view occupancy rates, revenue, tenant satisfaction, and complaint trends. Export reports as PDF or CSV.
Was this helpful?
Applications
How do I review dorm applications?
Go to Application to view all pending dorm applications from admins. Click on an application to view full details including dorm info, photos, and submission date. You can Approve, Request Changes, or Reject the application.
Was this helpful?
What criteria should I use to approve/reject a dorm application?
Check for:
  • Compliance with safety and health standards
  • Proper documentation and photos
  • Valid contact information
  • Adequate capacity and amenities listed
  • Clear description and pricing
Reject applications that lack critical information or violate standards. Use Request Changes for minor issues that can be corrected.
Was this helpful?
Can I request changes to an application?
Yes. Click Request Changes and provide detailed feedback on what needs to be corrected. The admin will be notified and can resubmit the application. This back-and-forth continues until the application meets requirements or is rejected.
Was this helpful?
How long do I have to review an application?
Applications should be reviewed within 5 business days of submission. The system sends reminders for pending applications. Use the Priority flag to mark urgent applications for faster review.
Was this helpful?
Complaints & Issues
How do I view and manage complaints?
Go to Complaints to view all system-wide complaints. Filter by property, status, or priority. Click a complaint to view details, tenant info, and attachments. You can assign it to an admin, add notes, and change status.
Was this helpful?
What is the complaint resolution process?
Complaints follow this workflow:
  • New: Recently submitted
  • In Review: Being investigated
  • Assigned to Admin: Delegated for resolution
  • Resolved: Completed and closed
  • Escalated: Requires higher authority
Update status as the complaint progresses through these stages.
Was this helpful?
How do I escalate a complaint?
If a complaint is complex or serious, click Escalate and select the escalation reason. Add detailed notes explaining why escalation is necessary. The complaint will be flagged for senior review and priority handling.
Was this helpful?
How do I generate a complaint report?
Go to Complaints → Reports. Select date range, property filter, and complaint category. Click Generate Report to create a comprehensive analysis including trends, common issues, and resolution times. Export as PDF.
Was this helpful?
Account & Settings
How do I change my account password?
Go to Settings → Security tab → Change Password. Enter your current password, then create a new one. Make sure your new password is at least 8 characters and includes letters, numbers, and symbols for security.
Was this helpful?
How do I update my account details?
Go to Settings → Profile tab. You can update your name, email, and contact information. Changes are saved immediately. Remember to click Save Changes if prompted.
Was this helpful?
What is my user role and what permissions do I have?
As a System Administrator, you have full access to:
  • All properties and properties management
  • Dorm application reviews
  • System-wide complaints and escalations
  • User and admin management
  • System reports and analytics
Your role cannot be changed through Settings — contact your organization administrator for role changes.
