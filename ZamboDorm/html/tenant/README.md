# Tenant Pages

Pages dedicated to tenant-facing features within the dormitory. These pages allow tenants to manage their accounts and interact with dorm services.

## File Overview

### Personal Information
- **tenant-myroom.html** - View and manage personal room information and details

### Financial Management
- **tenant-payment.html** - Payment tracking, billing history, and payment made/due information
  - Features: Alert box for upcoming payments, 3-column info cards, itemized billing breakdown, payment history table
  - Theme: Purple (#7c3aed) with yellow alerts (#f59e0b)

### Service Management
- **tenant-complaint.html** - Submit and track maintenance complaints and issues

### Visitor Management
- **tenant-visitors.html** - Manage and log visitor requests and entries

## Features

- Responsive design for mobile, tablet, and desktop
- Integrated with tenant authentication system
- Real-time payment status tracking
- Service request submission and tracking
